山东大学 计算机科学与技术学院云计算技术课程实验报告

学号：202018130168 姓名：黄薏霖 班级： 数据
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期： 2023.3.15
实验目的：熟悉个人博客系统的搭建。 具体包括：参考方案：注册 Github 账号，搭建 Hexo 环境并实现个人博客搭建，撰写实验报告。
硬件环境：联网的计算机一台
软件环境： Windows or Linux
实验步骤与内容： 1.安装两个本次所需要使用的软件：node.js 和 Git 2. 注册 Github 账号以及建立仓库 3. 接下来进入本地配置安装 Hexo 4. 设置 ssh 5. 上传测试博客 6.上传实验报告为内容的博客
结论分析与体会：这次实验的目的主要是熟悉个人博客系统的搭建。通过这次实验，我熟悉了 hexo 的一些常用命令，创建了一篇名为 first 的文章，并成功 部署到了 Github 上。此外，我了解到了 GitHub Pages 是由 GitHub 官方提供的一种免费的静态站点托管服务， 让我们可以在 GitHub 仓库里托管和发布自己的静态网站页面。Hexo 是一个快速、简洁且 高效的静态博客框架，它基于 Node.js 运行，可以将我们撰写的 Markdown 文档解析渲染 成静态的 HTML 网页。我还了解到了 Hexo + GitHub 文章发布原理：在本地撰写 Markdown 格式文章后，通过 Hexo 解析文档，渲染生成具有主题样式的 HTML 静态网页，再推送到 GitHub 上完成博文 的发布。总之，在这次实验中，我注册了一个 Github 账号，搭建了 Hexo 环境并实现了个人博客搭建。
